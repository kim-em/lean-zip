# Start Autonomous Work

Follow the Autonomous Work Cycle in `.claude/CLAUDE.md`:

1. Orient — `coordination orient`, read `progress/`, `VERIFICATION.md`
2. Plan — create branch, write plan, `coordination plan`
3. Execute — implement, review, or self-improve
4. Verify — final build+test, review diff, `/second-opinion`
5. Publish — write progress entry, create PR via `coordination create-pr`
6. Reflect — run `/reflect`, update `.claude` if needed

Begin by orienting.
