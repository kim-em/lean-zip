## Key decisions

- Removed `private` from `parseCompressedLiteralsHeader` to enable proof reasoning
  about header byte counts from the spec file. The function was opaque across module
  boundaries, preventing the proof from establishing `headerBytes ≥ 3`.
- Factored the headerBytes bound into a separate helper lemma rather than inlining
  the proof, keeping the main theorem clean.
- Used destructured arguments in the helper lemma (`regen, comp, hdr, fs` instead of
  `v : Nat × Nat × Nat × Bool`) to avoid tuple projection reduction issues with omega.
