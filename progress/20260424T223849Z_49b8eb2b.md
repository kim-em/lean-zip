## Decisions / patterns

- Reused the existing `write()` helper in
  `scripts/build-cd-lh-mismatch.py` rather than creating a sibling
  builder script — follows the precedent of
  `cd-empty-entry-crc-nonzero.zip` (one extra `write(...)` call with
  `lh_method=8, cd_method=8, lh_comp=0, cd_comp=0, lh_uncomp=42,
  cd_uncomp=42, lh_crc=0, cd_crc=0, payload=b""`). No new kwargs
  needed; the existing defaults cover everything.
- Picked guard shape `if uncompSize > 0 then unless compSize > 0 do …`
  to mirror the `uncompSize == 0 / crc == 0` shape of PR #1857 above
  — makes the ordering-disjoint `uncompSize > 0` / `uncompSize == 0`
  split visible at the `if` line.
