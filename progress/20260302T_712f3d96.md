### Notable discoveries

- **Array vs List**: `lengthBase`, `distBase` etc. are `Array Nat`, not `List`.
  `List.length_cons/nil` don't help with `.size`; using `have : T.size = N := rfl`
  lets the kernel evaluate the concrete size, then `omega` handles the rest.
- **Dependent getElem**: `rw [hlenSum]` fails when the bound proof differs
  between the hypothesis and goal. `subst` works because it replaces the variable
  entirely, sidestepping the dependent type issue.
- **`simp only` and constructor discrimination**: `simp only` does NOT close
  `none = some _` goals — need explicit `exact nomatch henc` or `contradiction`.
