## Patterns learned

1. **Concrete array size computation**: `#[...].size` needs `List.size_toArray` (not `Array.size_toArray`),
   then `List.length_cons` + `List.length_nil` to reduce to a number for omega
2. **Option do-notation in simp only**: `Option.pure_def, Option.bind_eq_bind, Option.bind_some` is the
   standard kit for unfolding Option monadic operations
3. **UInt16 normalization via dsimp**: `dsimp only [Nat.toUInt16]` + `exact` avoids linter false positives
   that occur when `Nat.mod_eq_of_lt` appears as both a simp arg and the final closing step
