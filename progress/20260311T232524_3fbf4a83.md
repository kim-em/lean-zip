## Decisions

- **Windowed mode restriction** (`windowSize > 0`): Simplifies offset-reach argument
  (offset ≤ windowSize ≤ output.size). This is the common case for real Zstd
  compressed blocks. Non-windowed case can be added later if needed.
- Used `split`-based proof instead of `simp`-based guard elimination for cleaner
  interaction with the goal's `if-then-else` chain (avoids `dsimp [letFun]` issues).
