## Decisions and patterns

- **Stale issue counts**: Both files were already audited in the
  bare-simp campaign. Did not skip; the issue also asked for shared
  infrastructure review (a separate, valid axis).
- **Tactic macros for repeated proof tails**: The
  `set_option hygiene false in local macro ... : tactic` pattern
  from `InflateRawSuffix.bfinal_suffix_dispatch` works well for
  extracting 9+ line repeated tactics sharing names with the
  enclosing proof. Key gotchas: can't attach doc comments to
  `set_option ... in` blocks (use plain `--` comments); `·` bullets
  don't work inside `` `(tactic| ...) `` — use `next =>`.
- **Normalize before extracting**: when similar blocks differ by one
  line, introduce a local `have` to unify them first, then extract.
- **Dead helpers**: `uint8_nat_roundtrip` and
  `validLengths_toUInt8_roundtrip` in InflateComplete are defined
  but unused. Left in place per project rule "Do NOT remove a
  working proof".
