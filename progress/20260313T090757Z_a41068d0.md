## Patterns Used
- **Loop bound capture**: `for h : i in [:n]` directly provides `i < n` for indexing
- **Guard + break**: When loop bound doesn't match array size (e.g., iterating 512 on
  variable-size array), iterate over `.size` and `break` at the logical limit
- **Inline if-guard**: For one-off accesses at computed positions, `if h : idx < arr.size
  then arr[idx] else default` avoids API changes
