## Accomplished

Added three field characterization theorems for `parseBlockHeader` in `Zip/Spec/Zstd.lean`:

1. **`parseBlockHeader_lastBlock_eq`** — When `parseBlockHeader` succeeds, `hdr.lastBlock`
   equals whether bit 0 of the 24-bit header word is set.
2. **`parseBlockHeader_blockType_eq`** — When `parseBlockHeader` succeeds, `hdr.blockType`
   is determined by bits 1-2: 0→raw, 1→rle, 2→compressed.
3. **`parseBlockHeader_blockSize_eq`** — When `parseBlockHeader` succeeds, `hdr.blockSize`
   equals bits 3-23 of the 24-bit header word.

## Proof pattern

All three use the same structure: unfold → unfold_except → split if-guard → split match
on typeVal → in each success branch, `obtain ⟨rfl, rfl⟩` from the result equality then
close by `rfl` (or `simp_all` for blockType which needs hypothesis rewriting to eliminate
impossible typeVal values).
