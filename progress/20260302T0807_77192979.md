## Accomplished

### 1. Created `lean-zstd-patterns` skill (new)

Comprehensive skill covering 8 patterns distilled from ~10 Track E sessions:

1. **Backward bitstream reading** — MSB-first, sentinel bit init, contrast with DEFLATE LSB-first
2. **FSE table construction** — 3-pass build (place -1 symbols, distribute via stepping, compute numBits/newState)
3. **Mixed forward/backward reading** — forward BitReader for FSE distribution, backward for symbols, with byte-aligned hand-off
4. **Byte position threading** — `(result, newPosition)` tuples through nested parsing, block-end anchoring
5. **Sequence execution** — 3-entry offset history, dual modes for literalLength=0 vs >0, overlap-aware match copy
6. **Huffman tree descriptors** — direct (nibble-packed) vs FSE-compressed, weight-to-code mapping
7. **RFC 8878 constant validation** — magic numbers, frame header bit fields, variable-width count encoding
8. **Predefined FSE distribution tables** — default probability arrays and fixed accuracy logs
