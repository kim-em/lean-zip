- **Date**: 2026-03-09 UTC

## Accomplished

Added two frame-level content theorems to `Zip/Spec/Zstd.lean`:

1. **`decompressFrame_compressed_lit_then_raw_content`** — When a frame contains
   a non-last compressed block (numSeq=0, literals only) followed by a last raw
   block, the output equals `literals1 ++ block2`.

2. **`decompressFrame_compressed_lit_then_rle_content`** — Same pattern but block 2
   is a last RLE block instead of raw.

Both lift the existing block-level theorems (`decompressBlocksWF_compressed_literals_then_raw`
and `decompressBlocksWF_compressed_literals_then_rle`) to frame level, following
the established `decompressFrame_raw_then_rle_content` proof pattern.
