## What was accomplished

Added two composed completeness theorems for `decompressFrame` in `Zip/Spec/Zstd.lean`:

1. **`decompressFrame_succeeds_single_raw`** — proves `decompressFrame` returns `.ok` for
   frames containing a single last raw block, given:
   - `parseFrameHeader` success (intermediate level)
   - No dictionary, no content checksum, no declared content size
   - Byte-level block header conditions (reusing `decompressBlocksWF_succeeds_single_raw`)

2. **`decompressFrame_succeeds_single_rle`** — same pattern for RLE blocks, using
   `decompressBlocksWF_succeeds_single_rle`

## Proof strategy

Both proofs follow the same 6-step pattern (~10 lines each):
1. Obtain block-level success from byte-level conditions via existing composed theorems
2. Unfold `decompressFrame` and thread `parseFrameHeader` result
3. Eliminate dictionary check (`dictionaryId = none`)
4. Unfold `decompressBlocks` and rewrite with block success
5. Simplify checksum/size guards (`contentChecksum = false`, `contentSize = none`)
6. Close with existential witness
