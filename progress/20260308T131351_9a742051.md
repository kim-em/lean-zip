## Key patterns discovered

- `omega` cannot reason about UInt8/UInt16/UInt32 comparisons directly — always
  need an explicit `have : ... := hlen_pos` to convert to Nat before omega can
  use the hypothesis. This is a recurring pattern worth noting in skills.
