## Accomplished

Added three new block-level spec theorems to `Zip/Spec/Zstd.lean`:

1. **`decompressRawBlock_pos_eq`**: When `decompressRawBlock` succeeds,
   `pos' = pos + blockSize.toNat`. Proves raw blocks advance the input
   position by exactly the block size.

2. **`decompressRLEBlock_pos_eq`**: When `decompressRLEBlock` succeeds,
   `pos' = pos + 1`. Proves RLE blocks advance position by exactly 1
   (the single repeated byte).

3. **`decompressRawBlock_content`**: When `decompressRawBlock` succeeds,
   `result[i] = data[pos + i]` for all valid indices. Proves raw blocks
   copy input bytes verbatim. Uses `ByteArray.getElem_extract`.

All three follow the existing monadic unfolding pattern (unfold, simp
bind/pure, split on guard, nomatch error, obtain from success).
