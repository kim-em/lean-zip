## Accomplished

Added two theorems to `Zip/Spec/BitReaderInvariant.lean`:

1. **`readBit_pos_le_size`**: Fully unconditional — when `readBit` succeeds,
   `br'.pos ≤ br'.data.size`. The success of `readBit` implies the guard
   `pos < data.size` passed, so no hypothesis on the input reader is needed.

2. **`readBits_pos_le_size`**: Requires `br.pos ≤ br.data.size` for the
   `n = 0` base case (which returns the reader unchanged). Uses a private
   helper `readBits_go_pos_le_size` with induction on `n`.

Both theorems follow existing proof patterns in the file and close cleanly
with `omega` after case-splitting on the `readBit` branches.
