  `header.dictionaryId` is `some n` where `n ≠ 0`, return
  `Except.error "Zstd: dictionary decompression not supported (dictionary ID: {n})"`

  frames, which lean-zip cannot decode

  returns an error containing "dictionary"

  last checksum byte with XOR 0xFF, verifies error containing "checksum"

  content size = 10) with no block data; verifies an error is returned
