## What this wave accomplished

### 1. Adler-32 concrete-shape ladder — fully closed end-to-end

Post-#1698, the seven Adler-32 rungs are all closed-form: `_empty`,
`_singleton` (#1661), `_pair` (#1673), `_replicate_zero` (#1679),
`_replicate` (#1684), `_append` (#1641), and **`_combine` (#1698,
new this wave)**.

**The novel content in #1698** is the `updateList_separation`
Spec-side identity: for any prefix `xs` and suffix `ys`,
`updateList (a, b) (xs ++ ys)` is decomposed *across* the modular
reduction so the join's closed form is expressible without
re-traversing `xs ++ ys`. This is the analogue of zlib's runtime
`adler32_combine(adler1, adler2, len2)` formula, lifted to the Spec
layer with no integer overflow concerns. #1700's review §F flagged
two missing fixture coverage gaps (empty-`xs`; large-random
deterministic split) which #1712 closed inside the wave.

### 2. CRC32 concrete-shape ladder — closed at `_pair` (intentional terminus)

#1701 lands `Spec.checksum_pair` / `Native.crc32_pair`: the Spec
theorem expresses `checksum [b₁, b₂]` as `0xFF000000 ^^^
mkTable[i₁]'h₁ ^^^ mkTable[i₂]'h₂` for two derived indices. The proof
is a `bv_decide`-closed XOR identity, using `generalize` to avoid the
distinct-proof-term pitfall (codified separately by #1704). The
ladder now reads `_empty`, `_singleton` (#1692), **`_pair` (#1701,
new)**, `_append`.

**Why the ladder stops here.** #1708's review confirms #1701's
minimality and records the boundary: `_replicate_zero` / `_replicate`
need multiplicative structure on the CRC polynomial ring (CRC of
`n` zeros is `crc · x^(8n) mod p`, no closed Nat form), and
`_combine` needs the matrix-representation trick. None reachable from
the Adler-32 template. #1709's new skill records this "when NOT to
apply" boundary so future planners stop generating `_replicate*`
issues for CRC32.

### 4. CD/LH flags consistency check + uniform fixture triad (#1716)

#1716 threads the central-directory general-purpose bit flag word
through the `Archive.Entry` struct (new `flags : UInt16 := 0` field
populated in `parseCentralDir`), then enforces a CD/LH
flags-consistency check inside `readEntryData` *before* the
compressed-payload read. The check XOR-masks bit 3 (the
data-descriptor flag — the only bit legitimately allowed to diverge
between CD and LH) and rejects mismatch on the remaining 15 bits.

The vector closed: a CD with bit 11 (UTF-8 name) set and an LH with
bit 11 clear would previously trick downstream LH re-parsers into
decoding the name under Latin-1 fallback while the extractor used the
CD-derived strict-UTF-8 name — a known ZIP-smuggling payload
shape. The new 122-byte
`testdata/zip/malformed/cd-lh-flags-mismatch.zip` fixture is
recipe-identical to the pre-existing `cd-lh-method-mismatch.zip` and
`cd-lh-size-mismatch.zip` from #1554; the **fixture triad
(method/size/flags) is now uniform**: same byte budget, same
inline-Python builder shape, same `assertThrows`-style test block.
#1720's review confirms the structural correctness, including the
non-obvious choice that the flags check fires *outside* the
`usesDataDescriptor` guard (because bit 3 is XOR-masked, the check is
data-descriptor-invariant by construction).

## Patterns surfaced

- **Separation-formula pattern (#1698 `updateList_separation`).**
  A Spec-side equality decomposing a long-input update across the
  modular reduction so the join's closed form references prefix
  state + suffix-only quantities — no re-traversal of the prefix.
  Generalisable alternative to reverse induction for any
  monoid-with-state checksum that needs a two-block combine
  theorem. Worth a `checksum-ladder-template` follow-up if a third
  checksum (e.g. XxHash) joins.
- **Fixture-triad pattern for CD/LH consistency (#1716 closes it).**
  `method` / `size` / `flags` mismatches now share three uniform
  122-byte fixtures with identical inline-Python builders,
  `assertThrows` blocks, and error-substring style ("X mismatch
  between CD and local header for {label}"). Codify in
  `error-wording-catalogue` if name/version variants follow.
- **Paired-review cadence held under decomposition pressure.** All
  four pairs landed at 21–69 min vs prior wave's 78–117 min. Data
  point: when feature work is one self-contained rung (not a
  layered proof like `_replicate`), paired-review stays inside
  meditate #1651 §4's budget.
