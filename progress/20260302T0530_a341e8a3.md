## Accomplished

- Added Zstd block-level parsing to `Zip/Native/ZstdFrame.lean` (~89 lines):
  - `ZstdBlockType` inductive: raw, rle, compressed, reserved
  - `ZstdBlockHeader` structure: lastBlock, blockType, blockSize
  - `parseBlockHeader`: parses 3-byte little-endian block headers per RFC 8878 §3.1.1.2
  - `decompressRawBlock`: verbatim copy of blockSize bytes
  - `decompressRLEBlock`: single byte repeated blockSize times
  - `decompressBlocks`: block loop dispatching on type, accumulating output until lastBlock
