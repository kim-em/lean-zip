Converted `Inflate.decodeHuffman.go` from fuel-based recursion to
well-founded (WF) recursion, continuing Track C2 fuel elimination.

- Added `BitReader.bitPos` definition: `pos * 8 + bitOff`

- Runtime guards at each recursive call site check bit position advances
  and stays within bounds, providing hypotheses for `omega`

## Key decisions

- **Runtime guard approach**: Chose runtime guards over proving
  `decode_advances` — simpler, avoids circular dependency between
  Inflate.lean and BitReaderInvariant.lean
- **dataSize parameter**: Passed `br.data.size` as a parameter to `go`
  rather than using `br.data.size` directly, because `omega` can't
  derive `br'.data.size = br.data.size` without invariant lemmas
- **Sorry in decodeHuffman_go_append**: The old fuel-based proof was
  working but the WF version needs a different induction structure;
  left as sorry for a future session
