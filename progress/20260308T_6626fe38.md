## Accomplished

Added two API-level theorems to `Zip/Spec/ZstdFrame.lean`:

1. **`decompressZstd_single_skippable`**: When input contains a single skippable
   frame at position 0, `decompressZstd` returns empty output.
2. **`decompressZstd_skip_then_standard`**: When input starts with a skippable
   frame followed by a standard frame, `decompressZstd` returns only the
   standard frame's content.

Both are direct specializations of existing WF-level theorems with
`output = ByteArray.empty` and `pos = 0`.
