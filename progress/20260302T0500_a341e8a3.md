## Key proof pattern

All fuel-based `induction fuel` proofs were converted using:
```
suffices ∀ k (br : BitReader) ...,
    dataSize * 8 - br.bitPos ≤ k → ... from
  this _ br ... Nat.le.refl ...
intro k
induction k using Nat.strongRecOn with
| _ k ih => ...
  rw [Inflate.inflateLoop.eq_1] at h
  simp only [bind, Except.bind] at h
```

WF guards handled via `split at h` chains before IH calls.
