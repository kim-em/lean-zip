### 2. `compressUnsafe` / `compress` split design

The extern is declared `private opaque compressUnsafe (data : @&
ByteArray) (level : UInt8) : IO ByteArray` (no default level — the
public wrapper supplies it), with a docstring that explicitly notes
out-of-range values reach miniz_oxide unchanged and directs callers
to use `compress` instead. The public `def compress (data : @&
ByteArray) (level : UInt8 := 6)` carries the user-facing default and
the clamp expression `compressUnsafe data (if level > 9 then 9 else
level)`. Module-internal callers retain access to `compressUnsafe`
under the `private` qualifier; weakening that qualifier would be a
deliberate change visible in source review.

### 3. Smoke-test observability

- Skip-path: the three new assertions sit inside the existing
  `withMiniz "compress(big)"` block at line 40, so the
  `MINIZ_OXIDE_DISABLE=1` / missing-cargo path keeps cleanly
  skipping via the `"miniz_oxide: not built with Rust support"`
  substring match.
- Payload: `prng = mkPrngData 4096`, the deterministic xorshift32
  PRNG payload (default seed `2463534242`) used throughout
  `ZipTest/MinizOxide.lean` and the sibling bench-comparison tests;
  reproducible from `ZipTest/Helpers.lean` alone.
- Sharpness: `mzLevel10.beq mzLevel9` and `mzLevel255.beq mzLevel9`
  are both byte-identical assertions. Without the clamp, `level=10`
  would produce level-10 bytes (different from level-9), and
  `level=255` would silently wrap to level 0 (stored blocks) at the
  Rust shim — also byte-different from level 9. The PRNG payload's
  cross-level distinguishability (the dictionary does not saturate
  at any non-trivial level) makes adjacent-level outputs observably
  distinct, so the assertions are sharp witnesses of the clamp.

### 5. Pattern reusability

the **first instance in this codebase** — `grep -r "private opaque"
Zip/` returns only `Zip/MinizOxide.lean`, and `grep -r "Unsafe\|
private opaque" .claude/skills/` returns nothing. Documentation
lives only in the docstrings on `compressUnsafe` and the public
`compress`. Decision: **at N=1 site, defer skill extraction**. The
pattern is sufficiently self-explanatory at the call site that
lifting it to a skill would be premature abstraction; if a second
TCB-boundary clamp adopts the same shape (the issue body called out
future `decompress` size limits as a candidate), that PR is the
