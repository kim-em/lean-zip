## Decisions

- Used `exact nomatch hgo` (Lean 4 idiom) instead of `exact absurd hgo nofun`
  for closing `none = some _` contradictions — more concise.
- Kept `simp [readBitsLSB]` for header evaluation: converting to `simp only`
  requires 20+ lemmas including `reduceCtorEq` for boolean equality,
  and the result is fragile.
- Used `decide_cbv` despite "experimental" warning because it halves the
  required recursion depth. `native_decide` is forbidden in this codebase.
