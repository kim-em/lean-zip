- **Policy lives in *Current local guardrails*, not in a new
  paragraph above the *Why trusted* line.** The new bullet is a
  guardrail, not a trust-justification. Putting it in
  guardrails keeps the *Why trusted* paragraph shorter and lets
  the bullet end with the "drift surfaced advisorily by ŌĆ”" cross-link
  to the new script.
- **`#TBD-VERIFY-PR` placeholder resolution flow.** Followed the
  exact pattern used by PR #2378 (commit `f5e6bef`): three commits on
  the branch ŌĆö script, inventory edit with placeholder, then a
  one-line substitution to the real PR number after PR creation. The
  squash-merge collapses them into a single commit on master. The
  intermediate state has one warning from
  `scripts/check-inventory-links.sh` (the placeholder is a known-soft
  warning by design); after substitution the warning count drops to
  zero.
- **Pre-existing uncommitted off-limits files in this worktree.** The
  worktree had stale uncommitted modifications to `.claude/CLAUDE.md`
  (off-limits per the agent off-limits-files rule) and
  `.claude/skills/agent-worker-flow/SKILL.md`. Stashed both before

  Restored the stash at session end so the worktree state is
