## Accomplished

Proved two content-level theorems for Zstd sequence execution helpers
in `Zip/Spec/ZstdSequence.lean`:

1. **`copyBytes_getElem_lt`**: For `i < dst.size`,
   `(copyBytes dst src srcPos count)[i]! = dst[i]!`.
   Proof by induction on `count`, using `ByteArray.push_getElem!_lt`.

2. **`copyMatch_getElem_lt`**: For `i < buf.size`,
   `(copyMatch buf offset length)[i]! = buf[i]!`.
   Via private helper `copyMatch_loop_getElem_lt` (induction on `length - k`),
   then instantiation at `k = 0`.

Both proofs follow the same structure as the existing size proofs
(`copyBytes_size`, `copyMatch_size`) — induction on the WF recursion
measure, with `ByteArray.push_getElem!_lt` handling the push step.
