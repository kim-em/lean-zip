## Accomplished

Added two theorems to `Zip/Spec/ZstdSequence.lean`:

1. **`executeSequences_loop_cons`**: Single-step unfolding theorem that relates
   `loop (seq :: rest)` to `loop rest` with the intermediate state. Proves that
   when all four guards pass (literal bounds, offset ≠ 0, offset ≤ output size,
   window size), processing one sequence transforms state predictably via
   copyBytes + resolveOffset + copyMatch. Proof uses `rw [.eq_2]` to unfold the
   cons case, `simp` to eliminate the first guard, then `split` on each remaining
   `if` condition, closing impossible branches via the hypotheses.

2. **`executeSequences_loop_cons_output_size`**: Intermediate output size grows
   by exactly `literalLength + matchLength`. One-line proof composing
   `copyMatch_size` and `copyBytes_size`.
