## Termination proof approach

The termination measure is `data.size - off`. The outer guard
`if hoff : data.size ≤ off` provides `off < data.size` in the else
branch. A runtime guard `if hadv : newOff ≤ off` (always satisfied
since block processing advances position) gives omega the second
fact `off < newOff`, completing the proof via `Nat.sub_lt_sub_left`.

Key insight: using `if hoff : ... then .error ... else do ...`
(explicit if-else) instead of `do; if ... then throw ...` (do-notation
early throw) preserves the dependent hypothesis through monadic bindings.
