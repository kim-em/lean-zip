# 2026-06-02 06:08 UTC — feature (one-shot #2476): Track D D-7 shared LZ77 matcher

## What was accomplished

Closed #2476 (Track D D-7): share one LZ77 token stream between the fixed and
dynamic encoders at level ≥ 5. Equivalence refactor — byte-identical output,
removes one redundant matcher pass.

- `Zip/Native/DeflateDynamic.lean`:
  - Extracted `deflateDynamicBlock (data) (tokens)` — the old `deflateDynamic`
    body minus the `let tokens := lz77GreedyIter …` line. Dropped the unused
    `windowSize` parameter (it only fed the matcher).
  - `deflateDynamic` is now a one-line wrapper:
    `deflateDynamicBlock data (lz77GreedyIter data windowSize)`.
  - `deflateCompressed` level-≥5 branch now binds `let tokens := lz77GreedyIter data`
    once and feeds both `deflateFixedBlock data tokens` and
    `deflateDynamicBlock data tokens`.
- Proofs:
  - `deflateDynamic_spec` (`DeflateDynamicCorrect.lean`): added `deflateDynamicBlock`
    to the `unfold` (body unchanged).
  - `inflate_deflateCompressed` / `deflateCompressed_pad` / `deflateCompressed_goR_pad`
    (`DeflateRoundtrip.lean`): the level-≥5 branch now opens with
    `rw [show (let tokens := …; pickSmaller (deflateFixedBlock …) (deflateDynamicBlock …))
    = pickSmaller (deflateFixedIter data) (deflateDynamic data) from rfl]`,
    converting the shared-`let` form back to the old independent-matcher form
    (definitionally equal since `lz77GreedyIter` is deterministic), then reuses
    the D-1b proofs verbatim.

## Decisions / patterns

- **The `let` blocks `split`.** A non-dependent `let` in the goal (shown as
  `have tokens := …`) stops `split` from finding the inner `if`. Rather than
  zeta-reducing and rewriting `lz77GreedyIter → lz77Greedy` (which would break
  the dynamic branch, whose spec is stated over `deflateDynamic data` =
  `deflateDynamicBlock data (lz77GreedyIter data)`, *not* the `lz77Greedy` form),
  the cleanest fix was a single `rw [show … = <old form> from rfl]` at the top of
  each branch. This restores the exact pre-D-7 goal, so the existing proof bodies
  are reused with zero further change.
- Verified empirically that output is byte-identical: all 144 native rows in
  `bench/results/latest.json` have unchanged `out_size`.

## Measured delta (level-6 compress throughput, this machine)

- 1 MiB `text` 5.67 → 10.84 MB/s (1.91×), `cyclic` 5.88 → 11.70 (1.99×),

- Levels 0–4 unaffected (don't use the shared path).
